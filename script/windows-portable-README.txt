Zed 便携版 - 使用说明
====================

1. 解压此 zip 到任意目录（如 D:\Zed）

2. 启动方式（任选其一）：
   - 双击 zed.exe 直接启动
   - 运行 install.ps1 创建桌面快捷方式并添加到 PATH，之后可双击桌面图标或终端输入 zed 启动

3. 运行 install.ps1 的方法：
   - 右键 install.ps1 -> 使用 PowerShell 运行
   - 或在 PowerShell 中执行: .\install.ps1

4. 若无法启动，请确保：
   - 显卡驱动已更新至最新版本
   - 杀毒软件未拦截
